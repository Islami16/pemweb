1. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web?

CSS digunakan untuk mengatur tampilan dan gaya dari elemen HTML.

  • Mengatur layout dan desain visual, seperti warna, font, jarak antar elemen, ukuran, dan posisi.
  • Membuat tampilan konsisten di seluruh halaman website.
  • Meningkatkan user experience dengan tampilan yang menarik dan profesional.

JavaScript digunakan untuk interaktivitas dan logika:

  • Menangani event (klik, input, scroll)
  • Memanipulasi DOM secara dinamis
  • Validasi form sebelum dikirim

2. Bagaimana CSS dan JavaScript dapat diintegrasikan dalam pemrograman web?

CSS:

  • Inline CSS: <p style="color: blue;">Teks biru</p>
  • Internal CSS: 
    <style>
p { color: blue; }
</style>
  • External CSS: <link rel="stylesheet" href="style.css">

JavaScript:

  • Inline JS: <button onclick="alert('Hello!')">Klik Saya</button>
  • Internal JS: 
    <script>
alert("Halo Dunia!");
</script>
  • External JS: <script src="script.js"></script>

3. Keuntungan menggunakan JavaScript dan CSS

CSS:

  • Memisahkan desain dari konten
  • Konsistensi tampilan
  • Menghemat waktu

JavaScript:

  • Menambahkan interaktivitas
  • Logika berjalan di sisi klien
  • Mengubah konten secara dinamis

4. Keuntungan Menggunakan Bootstrap

  • Cepat dan Efisien
  • Responsive Design Otomatis
  • Desain yang Konsisten dan Profesional

5. Contoh Penggunaan Bootstrap

Fungsi: Form login untuk input email dan password pengguna.

Penggunaan Bootstrap:

  • form-control: membuat input lebih rapi
  • btn btn-primary: tombol biru
  • Responsif otomatis untuk HP dan desktop

Hasil: Form login modern, bersih, dan responsif tanpa banyak CSS.
